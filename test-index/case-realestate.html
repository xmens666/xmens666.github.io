REAL ESTATE TECH

房源信息多而不乱 Listings organized, not overwhelming 物件情報を整理して見やすく 地图找房让成交快人一步 Map Search closes deals faster 地図検索で成約をスピードアップ

客户背景：不动产中介"安居东京"。手握大量房源，但还在用微信发长图。客户想找"离高田马场车站徒步10分钟内"的房子，销售找房找半天，客户体验差。


ZenCode 方案：开发地图找房APP。集成 Google Maps API，支持"画圈找房"、"条件筛选（南向/宠物/二楼以上）"。支持 VR 看房链接跳转。

Client Background: Real estate agency "Anju Tokyo" had a large inventory but relied on sending long images via WeChat. When clients wanted properties within a 10-minute walk from Takadanobaba Station, agents spent ages searching — poor client experience.


ZenCode Solution: Built a map-based property search app. Integrated Google Maps API with "draw-to-search", advanced filters (south-facing / pet-friendly / 2F+), and VR viewing links.

クライアント背景：不動産仲介「安居東京」。豊富な物件情報を持ちながら、WeChatで長い画像を送るだけ。「高田馬場駅から徒歩10分以内」の物件を探すのに時間がかかり、顧客体験が悪かった。


ZenCode ソリューション：地図検索アプリを開発。Google Maps APIを統合し、「円を描いて検索」「条件絞り込み（南向き／ペット可／2階以上）」に対応。VR内見リンクにも対応。
